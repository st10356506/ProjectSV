# Will You Be My Valentine?

A fun, interactive Valentine's Day proposal website.

## How to Use

1. Open `index.html` in your web browser (double-click the file, or right-click → Open with → your browser).
2. Share the link if you host it online, or open it on your computer when you're ready to ask!

## Features

- **Interactive Yes/No** – The "Yes" button leads to a celebration. The "No" button runs away when you hover over it and shows playful messages.
- **Expandable timeline** – "Our story" section with clickable entries. Add your own photos and captions for relationship milestones.
- **Confetti celebration** – When she says yes, hearts and confetti fill the screen.
- **Sage green accents** – Soft green elements complement the rose palette.
- **Responsive design** – Works on desktop, tablet, and phone.
- **Personalizable** – Edit `index.html` to change the text, colors, timeline entries, or add photos.

## Adding Your Photos

1. Create an `images` folder in the same directory as `index.html` (if it doesn't exist).
2. Add your photos with these names: `photo1.jpg`, `photo2.jpg`, `photo3.jpg`, `photo4.jpg`.
3. Edit the timeline in `index.html` to change dates, titles, and captions to match your photos.

## Customization

Open `index.html` in a text editor to personalize:

- **Main question** – Change the text in the `<h1>` tag.
- **Subtitle** – Edit the `<p class="subtitle">` line.
- **Timeline** – Add, remove, or edit timeline entries. Each entry has a date, title, image path, and caption.
- **No-button messages** – Update the `noMessages` array in the JavaScript section.
- **Yes-screen message** – Edit the text in the `#yes-screen` section.
