# Add your photos here

Place your relationship photos in this folder with these names:

- `photo1.jpg` — First met / The beginning
- `photo2.jpg` — First date
- `photo3.jpg` — Favorite memory
- `photo4.jpg` — Us today

You can use .jpg, .jpeg, .png, or .webp. If you use different filenames, update the `src` attributes in `index.html` in the timeline section.
